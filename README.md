# Car Management System (Java MVC)

Web/Console application built using Java, Servlets & JSP (MVC architecture) for managing car information.

## Features / Main Menu

1. Add Cars
- Add new car information such as name, model, color, speed, and price.

2. Delete All Matching
- Delete all cars that match specific criteria.

3. Find All Matching
- Search and display all cars that match given conditions.

4. Erase All Data
- Remove all stored car data from the system.

5. List Cars
- Display a list of all cars in the system.

## Technologies Used

- Java
- MVC Pattern (Model, View, Controller)
- Servlets & JSP (if web version)
- Collections / Lists for data management

## Purpose

This project was created for learning and practicing Java MVC architecture and  CRUD operations.
It demonstrates the ability to structure applications, manage data, and implement menu-driven programs.
